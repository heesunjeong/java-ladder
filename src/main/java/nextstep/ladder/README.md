# 기능 목록
- 참여할 사람 이름을 쉼표(,)를 구분자로 입력받는다.
- 이름의 최대 길이는 5자이다.
- 사다리 높이를 숫자로 입력받는다.
- 사다리를 만든다.
  - 사다리는 라인을 높이만큼 가진다.
  - 라인에 포인트가 생긴다.
  - 라인은 연속되는 포인트를 가질 수 없다. (ex. |-----|-----|)
- 만들어진 사다리를 출력한다.
```
pobi  honux crong   jk
    |-----|     |-----|
    |     |-----|     |
    |-----|     |     |
    |     |-----|     |
    |-----|     |-----|
```
